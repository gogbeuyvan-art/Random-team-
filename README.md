# Compétition Aléatoire

Petit site statique qui permet de former des équipes aléatoires à partir d'une liste de noms (un par ligne).

Comment utiliser :
1. Ouvrez le fichier `index.html` dans un navigateur (double-clic ou glisser-déposer dans la fenêtre du navigateur).
2. Entrez les noms, un par ligne.
3. Cliquez sur "Générer les équipes".

Déploiement :
- Pour héberger avec GitHub Pages : créez un dépôt, ajoutez ces fichiers à la branche par défaut, puis activez GitHub Pages dans les réglages du dépôt (branche à publier : par défaut).
- Le site est totalement statique — aucun serveur requis.
